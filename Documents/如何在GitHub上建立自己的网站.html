如何在GitHub上建立自己的网站

  1. 注册GitHub账号
  2. 创建仓库
  3. 安装开发工具
  4. 创建网站文件
  5. 编辑网页内容
  6. 部署到GitHub
  7. 注意事项

三、安装开发工具

需要安装： Visual Studio Code 和 Git 

git config --global user.name "你的用户名"
git config --global user.email "你的邮箱"

二、创建仓库

正确仓库命名示例：

用户名：yourname
    仓库名：yourname.github.io

四、创建网站文件

基础HTML结构示例：

<!DOCTYPE html>
    <html lang="zh">
    <head>
        <meta charset="UTF-8">
        <title>我的网站</title>
        <link rel="stylesheet" href="css/style.css">
    </head>
    <body>
        <h1>欢迎来到我的网站</h1>
    </body>
    </html>

五、编辑网页内容

基础CSS示例（保存为css/style.css）：

body {
        background: #1a1a1a;
        color: white;
        font-family: 'Microsoft Yahei';
    }

    h1 {
        color: #4A90E2;
        text-align: center;
    }

六、部署到GitHub

Git命令流程：

# 初始化仓库
    git init

    # 添加所有文件
    git add .

    # 提交更改
    git commit -m "首次提交"

    # 关联远程仓库
    git remote add origin https://github.com/yourname/yourname.github.io

    # 推送到GitHub
    git push -u origin main
